#rereurl(an URL Shortener)

#Introduction
you can use to shorten your URL and copy it.


#Start Up

-clone this project
$ git clone

-enter path
$ cd rereurl

-install npm
$ npm install

-run project
$ npm run dev

-Sucess:Now you can use it !
express server is running on http://localhost:3000

-End
ctrl + c
